It is built on LAMP Stack and uses the core technologies i.e 
- `PHP` 
- `MYSQL`
- `Javascript` 
- `Jquery` 
- `Boootstrap`
- `CSS`
- `HTML` 

## Installation Steps 
- Navigate to **phpmyadmin** and **create a database** named `otrsphp`. After creating the database, import the `otrsphp.sql` file from the database folder present in the folder. 

- Open `Xampp` 
- Start `Apache` 
- Start `MYSQL` 
- Run `localhost:port/name_of_folder`
